MIDI Console: filter [: transform/action]

Enter a string containing note filter criteria. You can optionally enter a colon and transform the returned notes by semi-directly changing their attributes. There are a few handy functions that you can use too.

The console targets selected notes in the active MIDI editor, all notes in the active MIDI editor or all notes of all active takes in selected items in the arrange view. In that order.


Filter section - variables and functions available

all - all targetted notes (selected or not in MIDI editor etc) (or just leave filter section empty)

p - pitch (0 - 127)

c - channel (1 - 16)

v - velocity (0 - 127)

l - length (in quarter notes)

ts - time signature, string.. eg "3/4" (include quotations)

ts_num - tim sig numerator

ts_denom - time sig denominator

beat(x) - checks whether note is on beat X, with a bit of lee way for unquantized material

beat(x,range) - same as above, but whether note is within range beats either side of beat X.

ran(x,low,high) - check if X is within a range of values


Transform/action section - variables available to alter and functions available

p - pitch (0 - 127)

c - channel (1 - 16)

v - velocity (0 - 127)

l - length (in quarter notes)

leg() - legato


eg

v>50 and c==2 - will select and make available for transforming all notes with a velocity over 50 on channel 2. Remember the double equals in this filter section.

beat(2) or nn=="kick":c=10 - will select and make available for transforming all notes that are on beat 2 of a measure (+/- preset tolerance, so it catches slightly early and late notes) or the pitch is on a named note row called "kick". Or "Kick", or "kIcK" - it uses lowercase to make matching easier. Then changes channel to 10.
